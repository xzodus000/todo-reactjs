Clone the project

```bash
  git clone [https://link-to-project](https://github.com/xzodus000/todo-reactjs.git)
```

Go to the project directory

```bash
  cd todo-reactjs
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
